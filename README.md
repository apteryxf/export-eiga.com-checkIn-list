# export-eiga.com-watch-list

1. `git clone https://github.com/apteryxf/export-eigaDotCom-checkIn-list`

1. `yarn`

1. add `{projectRoot}/credential.json` for your account credential

1. `yarn start`

##

Here is sample of `credential.json`.
  ```json
    {
      "userId": "123456",
      "email": "foo@gmail.com",
      "password": "fooBarBuz"
    }
  ```
