# export-eiga-dot-com-watch-list

1. `git clone https://github.com/apteryxf/export-eigaDotCom-checkIn-list`

1. `yarn`

1. add `{projectRoot}/credential.json` for your account credential

1. `yarn start`

##

sample credential.json
  ```json
    {
      "userId": "123456",
      "email": "foo@gmail.com",
      "password": "foobarbuz"
    }
  ```
